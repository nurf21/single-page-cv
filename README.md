# Professional Curriculum Vitae (CV) - HTML Implementation

## Project Overview
This project is a single-page professional CV built with semantic HTML and external CSS, designed to showcase a junior frontend developer's skills, education, and work experience. The implementation follows modern web standards with a clear separation of concerns between HTML structure and CSS styling.

## Key Features
- **Semantic HTML structure** using proper sectioning elements
- **External CSS** for maintainable styling
- **SEO optimization** with comprehensive meta tags
- **Open Graph integration** for enhanced social media sharing
- **Responsive design** that works on all device sizes
- **Modern styling** with CSS variables and flexbox
- **Data URI favicon** for quick loading
- **Accessibility considerations** throughout the document

## Project Structure
```
index.html            # Main HTML file containing content structure
style.css             # Stylesheet for all visual presentation
README.md             # This documentation file
```

## Sections Included
1. **Personal Information** - Name, job title, and contact details
2. **Skills** - Technical and professional abilities with visual tags
3. **Education** - Academic background with achievements
4. **Work Experience** - Professional history with accomplishments
5. **Online Presence** - Links to professional profiles

## How to Customize
1. Replace placeholder text in index.html with your personal information
2. Update skills to match your expertise
3. Modify education and work experience sections
4. Add your actual social media profile links
5. Adjust the color scheme in the CSS variables in style.css
6. Modify any styles in style.css to match your preferences

## Technologies Used
- HTML5
- CSS3 (with variables and modern layout techniques)
- SVG (for favicon implementation)
